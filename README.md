## Install
Install required gems through `bundle`.  

```bash
bundle install
```

## Test
```bash
ruby spec/api_spec.rb
```

## Run
Run this API server using `puma` at project root.

```bash
puma
```

Server would listen on http://0.0.0.0:9292.
